# Iamunix | Security Research & Bug Hunting

Welcome to my personal security research blog. I write about my journey in bug hunting, focusing on mobile security, web vulnerabilities, and Web3.

🌐 **Live Blog**: [iamunixtz.github.io](https://iamunixtz.github.io)

## Featured Research

- **Voox Android ATO**: A deep dive into a 1-click Account Takeover (ATO) chain involving JSBridge exploitation and hardcoded secrets.
- **OTP Rate Limit Bypass**: Exploiting email case manipulation to bypass security controls on OTP delivery.
- **S3 Bucket Misconfigurations**: Identifying and exploiting insecure cloud storage.

## About the Blog

This blog is built using the [Jekyll](https://jekyllrb.com/) framework and the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme. It serves as a repository for my technical writeups and a record of my professional growth in cybersecurity.
